变频双驱动器，新版本软件变更说明：
时间：2021/06/03		版本：V1.2			作者：LHC

新增/修改：
1、修改了密码控件触发方式，默认无密码，进入系统后修改密码生效。
2、增加了系统参数存储到IAP_FLASH区的CRC校验方式，增加系统稳定性。
3、修改了串口驱动，并加入了多串口通讯模型。
4、修改定时器底层驱动。
5、修改了变频器数据存储PLC方式。由原来的逐条存取，变换为块地址连续存储。

工作流程：
①首先串口3通过485协议定时（1S）向变频器请求数据，最终变频器回复的数据将会暂存到RS485缓冲区；
②定时对接收数据通过ModbusRTU协议处理（每条数据200ms）后，通过串口4（STC与STM32直连串口）发送给PLC固定区域存储；
③云端1min钟发送一条请求命令向PLC询问数据，串口2（4G模块）通过多串口协议获得相应数据。

特别说明：
①串口屏幕（迪文屏）直连PLC串口；PLC中STM32程序是固化程序，无法修改。
②屏幕数据呈现方式和实际读数据方式相反；显示速度较慢原因有：屏幕本身问题、PLC取出数据速度、PLC数据存入速度等。
